## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)

### Installation
1. Clone this repository:
   ```bash
   git clone <repo-url>
   cd chat-booba
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the server:
```bash
node server.js
```

Open your browser and go to [http://localhost:3000](http://localhost:3000) (or the port specified in your server).
